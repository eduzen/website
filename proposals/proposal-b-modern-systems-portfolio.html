About

I'm Eduardo Enriquez, a senior backend engineer based in Amsterdam. I've spent over 12 years building software — mostly in Python and Django, with deep experience in system architecture, API design, and making legacy codebases better.

Before engineering, I studied philosophy at the University of Buenos Aires. That background shapes everything: how I think about problems, how I teach, and why I value clarity in code as much as correctness.

I consult for companies and individuals, teach Python and Django privately, and write about the craft of engineering. I care about code that works, teams that communicate, and systems that last.
